1) Инициализировать проект 
    npm init

2) Установить библиотеку express
    npm install express

3) Написать код в index.js, чтобы сайт открылся по адрессу: localhost:3000
    а) использовать app.use()
    б) использовать express.static(...)

4) Вывести в консоль какую-нибудь информацию
    a) использовать app.use()
    б) вывести номер запроса
    в) вывести request.url

5) Изменить страницу <Моя группа> на своё усмотрение

6) Написать что-нибудь о себе в раздел <Обо мне>
    a) Разблокировать кнопку (убрать disable)
    б) Дописать что-нибудь в файл about.html
